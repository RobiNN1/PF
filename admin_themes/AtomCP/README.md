# AtomCP Admin Theme

![Version](https://img.shields.io/badge/Version-2.0.0-blue.svg)

Icons by [Icons8](https://icons8.com/).

**Features**

- Light and Dark mode
- Ajax Admin Chat
- Custom icons
- Charts
- Notes (each admin has own)
- Stats panel
- Online users panel
- Debug bar

Theme requires PHPFusion 9.03.100 and above.
