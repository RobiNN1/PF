# AtomCP Admin Theme

![Version](https://img.shields.io/badge/Version-2.0.0-blue.svg)

The most advanced admin theme for PHPFusion.

## Features

- Light and Dark mode
- Ajax Admin Chat
- Custom icons
- Charts
- Notes (each admin has own)
- Stats panel
- Online users panel
- Debug bar

## Requirements

- PHPFusion >= 9.03.100
